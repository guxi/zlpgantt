使用说明

可用以下三种方式加载数据

  • 一. 文件-示例
  • 二. 拖动数据文件到左边栏
  • 三. 文件-打开，选择写好的数据文件
  • 注： 数据文件可以是JSON文件（如下格式）或Excel表（参见例表）

数据文件格式

      
       {
          "title": [
          "项目","开始时间","结束时间"
          ]
          "data": [
          "a" , 43.3 , 17.8
          ],
            ...
          [
            ...
          ]          
        }
